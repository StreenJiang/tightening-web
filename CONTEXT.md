# CONTEXT.md — tightening-web

> 领域术语词汇表。不含实现细节。

## 核心概念

| 中文 | 英文 | 定义 |
|------|------|------|
| 工作台 | Workplace | 登录后的首页总览，汇总各模块关键数据 |
| 任务管理 | Mission | 拧紧任务的创建、下发、追踪与结果归档 |
| 站点 | Workstation | 产线工位的拧紧执行界面，操作工按引导完成拧紧 |
| 数据分析 | Analysis | 拧紧结果的追溯查询、SPC 统计、报表 |
| 设备管理 | Device | 拧紧枪/控制器的注册、校准、维保记录 |
| 系统集成 | Integration | 与 MES（工单同步）、PLC（产线控制）的对接 |

## 角色

| 中文 | 英文 | 职责 |
|------|------|------|
| 操作工 | Operator | 在工位上执行拧紧任务 |
| 工艺工程师 | Process Engineer | 定义拧紧策略和工艺参数 |
| 管理员 | Admin | 系统配置、设备管理、用户管理 |

## 部署拓扑

- **工控机（本地）：** N2840 主板 + 4GB DDR3 + 64GB SSD，运行收紧流程前端应用（浏览器打开 localhost）
- **中心服务器（远程，后续）：** MES 服务器，工控机通过内网与之交互
